# AdobeBrand Concierge

Adobe Brand Conciergeは、AI を活用して、企業と Web サイト訪問者の関わり方を変えるデジタルコンパニオンです。 一般的なチャットボットとは異なり、Brand Conciergeは各訪問者の意図に合わせてパーソナライズされた対話型のエクスペリエンスを提供し、製品の検出、オプションの比較、即座の回答の取得、ガイド付きのお勧めのリアルタイム受信を支援します。 プラットフォームは、B2C と B2B の両方に対応し、あらゆるデジタルチャネルでブランドのインテリジェントな拡張として機能し、ブランドの声、コンテンツの整合性、コンプライアンスを維持します。

>[!VIDEO](https://video.tv.adobe.com/v/3444149/?learn=on&enablevpops)

## 仕組み

Brand Conciergeは高度な AI を使用し、Adobe ツールと連携して、各訪問者に合わせて調整された役立つリアルタイムの会話を提供します。 承認済みのブランドコンテンツ、製品カタログ、その他の信頼できるソースを活用し、すべての情報がブランドにとって安全で最新であることを保証します。

* 対話型ジャーニー：訪問者は、自然なテキストを使用してやり取りし、質問したり、視覚的な製品の比較を受けたり、カスタマイズされたレコメンデーションを調べたりします。
* オムニチャネル統合：Brand Conciergeは、ブランドのデジタルチャネル（web サイト、モバイルアプリ、カスタムデプロイメント用の統合 SDK など）に埋め込むことができます。
* ブランドコントロールとガバナンス：マーケターは、エージェントの動作、メッセージング、ワークフローロジックを設定するための簡単でローコードのツールを使用して、ブランドのトーンと安全性を常に前面と中央に保つことができます。
* 分析と最適化：組み込みのダッシュボードは、エンゲージメント、フィードバック、コンバージョン指標を可視化し、エクスペリエンスを継続的に絞り込むことができます。

## Web サイトの所有者にとって特別で有益な理由

Adobe Brand Conciergeは、いくつかの点で従来のチャットボットやサードパーティアシスタントとは異なります。

* コンバージョンとロイヤルティの向上：Brand Conciergeは、静的な検索ページや FAQ ページではなく、即時に関連性が高くパーソナライズされた回答で訪問者を満たすことで、意思決定を促進し、コンバージョンを促進し、所有するプロパティにユーザーを集中させ続けます。
* 近日公開予定 – シームレスな人間による引き継ぎ：質問にライブタッチが必要な場合は、Brand Conciergeが（コンテキストを使用して）話し合いを営業またはサポート担当者に引き継ぐことができます。これにより、継続性とスムーズなカスタマーエクスペリエンスを確保できます。
* 継続的な学習と改善：各インタラクションは、顧客プロファイルを充実させ、実用的なインサイトを提供します。 マーケターやサイト所有者は、顧客が何を求めているかを確認し、フィードバックを収集し、フローを最適化できます。これらはすべて、開発者を必要としません。
* 柔軟なデプロイメント、迅速な価値実現：Brand Conciergeをウィジェットとして簡単に追加したり、サイトやアプリ全体でSDKを介して埋め込んだりできます。 Adobeとの深い統合により、AEM、AEP、Commerceを使用するブランドの迅速な設定が可能になりますが、それらなしで機能することもできます。

Adobe Brand Conciergeを採用することで、web サイトの所有者は、かつてないレベルのデジタルエンゲージメントを可能にします。つまり、関心のある瞬間を測定可能なビジネス結果に変える、高品質でブランドに合わせた会話で、訪問者をガイドすることです。
