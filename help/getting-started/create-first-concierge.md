# 最初のコンシェルジュを作成

Adobeを初めて導入し、パーソナライズされた会話体験を提供する方法を学びましょう。 コンシェルジュの名前を付ける方法、役に立つコンテンツを追加する方法、コンシェルジュの対応を選択する方法、自社のブランドに合わせてコンシェルジュの声を聞く方法などについて解説します。 準備ができたら、テストして、デスクトップとモバイルでの動作を確認できます。

## この動画は誰のためのものでしょうか？

* デジタルExperience Manager
* マーケティング技術者
* Adobe Customer Experience Designer

## ビデオコンテンツ

* Adobe Brand Conciergeの設定方法
* コンテンツソースを連携させて正確に回答する方法
* ブランドボイスをカスタマイズし、エクスペリエンスをテストする方法
