# データストリーム IDについて説明します

データストリーム IDが、サイトのWeb SDK イベントをAdobe Experience PlatformのAdobe Brand Conciergeに接続する方法を説明します。 IDが必要な理由、データを適切なサンドボックスやデータセットにルーティングする方法、開発、ステージング、実稼動のために別々のデータストリームを使用する方法などについて説明します。

## この動画は誰のためのものでしょうか？

* 開発者
* Product Manager
* Solution Consultant

## ビデオコンテンツ

* Web SDKとBrand Conciergeにデータストリーム IDが必要な理由
* データストリームがサンドボックス、データセット、スキーマ、IDにイベントをルーティングする方法
* 開発、ステージ、制作で異なるデータストリームを使用する理由
